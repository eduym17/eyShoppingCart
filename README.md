## Getting Started

To get a local copy and run this project follow the next steps.

### Prerequisites

npm
  ```sh
  npm install
  ```

### Setup

  ```sh
  $ git clone https://github.com/eduym17/EY-shopping-cart
  ```
  
  ```sh
  cd main
  ```

### Install

  ```sh
  npm install
  ```

### Usage

  ```sh
  npm npm start
  ```

### Run tests

  ```sh
  npm test
  ```
  
### Run build

```sh
npm run build
```


### Deployment

Make a pull request and when approved and merged the changes will show up on website.
